Hotkey Tabs has been updated

New Features:

  • Select last open matching page

    When multiple open pages match the search term, the option is now available to select the most recently opened tab.

  • RegEx matching now supported

    You can now match tabs with Regular Expressions while specifying the url to visit if no match is found.

  • Detects if current selected tab is blank

    If the active tab is a blank new tab page, instead of opening the page in a new tab the page will be opened in the current tab.

Important Changes:

  • Tab switching reverted to version 0.1.5 style

    'Switch/Open Site' will match any url that contains the search url within it.
    'Switch/Open page' will only match a url that is an exact match of the search url.
